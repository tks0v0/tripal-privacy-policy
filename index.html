Tripal プライバシーポリシー 最終更新日: 2026-07-18 1. 概要（基本方針） Tripal はローカルファーストの旅行計画アプリです。アカウントを作成せず、 Supabase 設定を注入せず、ログインしない場合は、旅行データを端末内に保存して 利用できます。クラウド機能は任意であり、利用者が Email Magic Link でログインし、 同期または共有機能を利用する場合にのみ使用されます。 2. 取得情報 ローカル利用時（ログインしない場合）に端末内へ保存される情報: 旅行名、日程、場所、予定、持ち物、費用、メンバー表示名、個人予算などの入力内容 アプリ設定および端末内のログイン状態に関する情報 これらは端末内の Hive データベースに保存されます。完全ローカルモードでは、 Tripal はこれらの旅行データをクラウドへ送信しません。 任意で Email Magic Link によるログインを利用する場合、次の情報が Supabase に送信・保存・処理されます。 メールアドレス（認証メールの送付、アカウント識別のため） Supabase Auth が発行するアカウント識別子およびセッション情報 同期または共有対象として利用者が保存した旅行データ（旅行、メンバー、予定、費用、個人予算、および関連する識別子・日時） 3. 利用目的 Supabase は、任意ログイン、Magic Link の送信、セッション管理、旅行データの 保存・同期・共有のために使用します。広告 SDK、行動分析 SDK、クラッシュ解析 SDK、 位置情報 SDK は本バージョンに含めません。Tripal は個人データを販売しません。 4. 第三者提供 共有旅行の参加者は、その旅行を利用するために必要な旅行情報を閲覧できます。 個人予算は、設計上、本人かつ当該旅行の参加者だけが閲覧・更新できるように制限されます。 これ以外の第三者への旅行データの提供は行いません。 5. データの保存期間 端末内データは、アプリ内で削除するか、アプリのデータを消去またはアンインストールするまで端末に残ります。 同期済みデータは、アプリの削除操作と同期仕様に従って論理削除される場合があります。 6. セキュリティ アプリは、任意の Supabase 機能を利用する場合に HTTPS で Supabase と通信します。 Android アプリはインターネット通信の権限のみを使用します。位置情報、カメラ、連絡先、 ストレージなどの危険な権限は要求しません。 7. 利用者の権利／アカウント・データ削除依頼 現在、Supabase Auth アカウントをアプリ内から完全削除する画面はありません。 アカウントおよび同期データの削除を希望する場合は、下記の問い合わせ先までご連絡ください。 本人確認のうえ、合理的な期間内に対応します。 8. 子どものプライバシー 本アプリは年齢確認機能を提供しません。対象年齢・保護者同意など、配信先ストアで必要な 情報は Play Console の対象年齢・コンテンツ評価へ正確に申告します。 9. 改定 このポリシーを変更する場合は、変更内容と発効日をこのページに掲載します。クラウド機能、 収集データ、送信先に重要な変更がある場合は、必要に応じて利用者へ通知します。 10. 問い合わせ先 アカウント・データの削除依頼、その他プライバシーに関するお問い合わせは 下記までご連絡ください。 zd.neuralstudio@gmail.com Tripal
